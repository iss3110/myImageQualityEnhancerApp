# myImageQualityEnhancerApp
prend une image couleur .jpg, et la segmente, càd regroupe les pixels qui se ressemblent (càd qui ont à peu près la même couleur) avec un paramètre qui modifie le seuil de "ressemblance de pixel" , qui remplace la valeur des pixels de chaque sous-groupe avec la moyenne des valeurs des pixels de ce même sous-groupe
